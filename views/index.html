Greetings, traveller. To use this API is fairly simple and straight forward.

Just visit https://lotr-random-quote-api.herokuapp.com/api/quote and you will get a random Lord of The Rings quote in the following format:

{

"quote": "Torment in the dark was the danger that I feared, and it did not hold me back.",

"author": "Gimli",

"image": "https://lotr-random-quote-api.herokuapp.com/images/gimli.jpg"

}

If you want to see an example of this API being used, here is a simple React App you can check out.
